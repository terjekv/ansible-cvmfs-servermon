# Description

Deploy [Cern's CVMFS prometheus exporter](https://gitlab.cern.ch/cloud/cvmfs-prometheus-exporter) to a system using ansible.

# Requirements

- Ansible >= 2.7 (It might work on previous versions, but we cannot guarantee it)
